Pacman

Navigate through the maze, eat all dots while avoiding ghosts!

Game Information

Pacman is one of the most iconic arcade games ever created. The objective is to eat all the dots in the maze while avoiding four colorful ghosts that pursue you. Power pellets allow you to temporarily eat the ghosts for bonus points.

Originally released in 1980, this modern web adaptation brings the classic gameplay to your browser with all the charm of the original.

How to Play:

  • Use arrow keys to control Pacman's movement
  • Eat all dots to complete a level
  • Avoid ghosts - if they catch you, you lose a life
  • Eat power pellets to temporarily hunt ghosts
  • Eat fruits for bonus points

Sudoku

Challenge your logical thinking with this addictive number placement puzzle!

Game Information

Sudoku is a logic-based number placement puzzle. The objective is to fill a 9√ó9 grid with digits so that each column, each row, and each of the nine 3√ó3 subgrids contains all of the digits from 1 to 9 without repetition.

The puzzle provides a partially completed grid and challenges the player to complete the remaining cells.

How to Play:

  • Click on an empty cell to select it
  • Type a number from 1-9 or click on the number buttons
  • Each row, column, and 3√ó3 box must contain numbers 1-9 with no duplicates
  • Use logic to determine which numbers go where
  • Complete the entire grid correctly to win

Minesweeper

Clear the minefield without detonating any bombs!

Game Information

Minesweeper is a classic puzzle game that challenges your deductive reasoning and memory. The goal is to clear a rectangular board containing hidden mines without detonating any of them, using clues about the number of neighboring mines in each field.

This game first became popular as part of Microsoft Windows in the 1990s and has remained a favorite brain teaser ever since.

How to Play:

  • Left-click to reveal a cell
  • Right-click to flag a potential mine
  • Numbers indicate how many mines are adjacent to that cell
  • Use logic to deduce where mines are located
  • Clear all non-mine cells to win

Snake

Control a hungry snake and eat as much food as possible without crashing!

Game Information

Snake is a timeless arcade game where you control a growing snake that must eat food while avoiding collisions with walls and itself. Each time the snake eats food, it grows longer, making the game progressively more challenging.

This game gained immense popularity in the late 1990s when it was pre-installed on Nokia mobile phones, becoming one of the first widely played mobile games.

How to Play:

  • Use arrow keys to change the snake's direction
  • Guide the snake to eat the food items
  • The snake grows longer each time it eats
  • Avoid hitting walls or the snake's own body
  • Try to achieve the highest score possible

2048

Combine matching numbers to reach the elusive 2048 tile!

2048 The 2048 game requires implementation from the GitHub repository. View on GitHub

Game Information

2048 is an addictive sliding puzzle game created by Gabriele Cirulli in 2014. The objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

The game became an overnight viral sensation due to its simple yet challenging gameplay, inspiring numerous clones and variations.

How to Play:

  • Use arrow keys to slide all tiles in one direction
  • When two tiles with the same number touch, they merge into one tile with double the value
  • After each move, a new tile with a value of 2 or 4 appears randomly on the board
  • Continue combining tiles to reach the 2048 tile
  • The game ends when there are no more possible moves
